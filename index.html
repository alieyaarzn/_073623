About Me

Hello! I'm Alieya Ruzaini, a passionate web developer focused on creating functional and interactive dashboards using real-time APIs. This project demonstrates the integration of public APIs for visualizing useful data through two dashboards:

  • Holiday Dashboard: View upcoming public holidays globally using Nager.Date API.
  • Weather Dashboard: Check live weather updates by city using OpenWeather API.
